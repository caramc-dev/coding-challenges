## Concepts Covered

- **Ternary** – inline if/else expressions
- **Modulus** – `%` for remainders and divisibility checks
- **Conditional branching** – if/elif/else chains
- **String formatting** – f-strings, `.format()`, `.join()`
- **String manipulation** – slicing, `str()`, `.split()`
- **List comprehension** – one-liner list building with conditions
- **Loops** – for loops, `enumerate()`, building lists iteratively
- **Sorting** – `sorted()`, slicing sorted results
- **Maths** – unit conversion, arithmetic operations
